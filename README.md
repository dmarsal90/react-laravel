## Run Locally

Clone the project

```bash
  git clone https://github.com/dmarsal90/react-laravel.git
```

Go to the project directory

Navagate to the backend and frontend folder, install dependencies and start the server

For the backend
```bash
  cd api 
  composer install
  php artisan serve 
```
For the frontend
```bash
  cd front
  npm install
  npm run dev
```




## Deployment

To deploy this project run

```bash
  npm run deploy
```
